GUIDELINES

Guidelines for extracting cognitive behavioral data from DATA LAB page
  • Browse MouseBytes: www.mousebytes.ca.
  • Click on "DATA LAB" from TOUCHSCREEN tab in the main menue.



  • An Example for selecting the values from the drop down lists for 5SCRTT task.
    • Select "Cam 5_Choice_V3" from the first list.
    • Select "FB_5Choice" and "DP_5Choice" from the Experiment list.
    • Select "Cam Mouse 5C Touch Var 1" from the Session list.
    • Select "Schedule Name" from the key feature list.
    • Select the following features: "Threshold – Accuracy %", "Threshold – Omission %", "Trial Analysis – Correct Response Latency", "Reward Collection Latency", "Premature Response - Total", "Perseverative Correct -Total". from the list. Note that when a feature is selected from this list, a new panel "Animal Data" gets visible.
    • Select "MEAN" and "COUNT" from the aggregation function list.
    • Select "Dr. Marco Prado – Robarts Research Institute" and "Dr. Boyer Winter - University of Guelph" from PI and Institution list.
    • You can filter animal data like age, sex, strain, and genotype from "Animal Data" panel. However, If no value is filtered from this panel, everything will be automatically selected and returned in the result.
    • Once everything is properly selected "Get Data" button will get enabled, and you can click on it to get data and browse it in a table.
    • When the result is loaded, click on "Export Data" button to download the result in CSV format. You can also generate a link for the obtained result by clicking on "Generate Link" button and share the data with others.



  • The same steps should be followed for PAL and PD tasks. You can select the following values from the drop down lists for PAL:
    • Select "PAL_V3" from the first list.
    • Select "FB_PAL" and "DP_PAL" from the Experiment list.
    • Select "PAL Analysis" from the Session list.
    • Select "Schedule Name" from the key feature list.
    • Select the following features: "End Summary - % Correct", "End Summary – No Correction Trials", "Correct Touch Latency", "Correct Reward Collection" from list.
    • Select "MEAN" and "COUNT" from the aggregation function list.
    • Select "Dr. Marco Prado – Robarts Research Institute" and "Dr. Boyer Winter - University of Guelph" from PI and Institution list.

  • You can select the following values from the drop down lists for PD:
    • Select "PD_V3" from the first list.
    • Select "FB_PD" and "DP_PD" from the Experiment list.
    • Select "PD Analysis" from the Session list.
    • Select the following features: "End Summary % Correct", "End Summary – No Correction Trials", "Correct Touch Latency", "Correct Reward Collection"
    • Select "MEAN" and "COUNT" from the aggregation function list.
    • Select "Dr. Marco Prado – Robarts Research Institute" and "Dr. Boyer Winter - University of Guelph" from PI and Institution list.
